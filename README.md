# Docker PHP

**Docker PHP** (for lack of a better name) is a [Docker](http://docker.com/) client written in PHP.
This library aims to reach 100% API support of the Docker Engine.

Note: this is a hard fork of [`docker-php/docker-php`](https://github.com/docker-php/docker-php) which became unmaintained
since 2019. The fork was done in January 2026. All copyright for previous work belongs to the original authors,
full Git history is retained.
